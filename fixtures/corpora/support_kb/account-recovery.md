# Account recovery

## Reset a password

Customers can request a password reset from the sign-in page. The reset link expires after 30 minutes.

## Restore an archived workspace

Workspace owners can restore an archived workspace for 14 days. Support must verify the workspace ID and owner email before restoration.

## Reset a password

Customers can request a password reset from the sign-in page. The reset link expires after 30 minutes.

## Delete an account

Account deletion is permanent after the seven-day cancellation window. Workspace export must finish before deletion begins.
